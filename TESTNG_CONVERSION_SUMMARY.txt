     Tests adding a product to the shopping cart
   
   ✓ @Test testVerifyProductInCart() - Priority 3
     Tests verification of product in cart after navigation

KEY IMPROVEMENTS:
=================
✓ Tests can be run individually or as a suite
✓ Setup/Teardown automated with @BeforeClass/@AfterClass
✓ Inheritance-based code reuse (extends BaseTest)
✓ ConfigReader automatically initialized in BaseTest
✓ Test prioritization with @Test(priority = N)
✓ Test descriptions with description parameter
✓ Proper exception handling
✓ Assertions for validation
✓ Easy to maintain and extend


HOW TO RUN TESTS:
=================

Option 1: Run from testng.xml (Right-click testng.xml → Run As → TestNG Suite)

Option 2: Run individual test (Right-click LandingpageBase.java → Run As → TestNG Test)

Option 3: Run from Maven command:
mvn test -Dsuite=testng.xml


TEST EXECUTION FLOW:
====================
1. @BeforeClass → Launches Chrome browser & initializes ConfigReader
2. @Test(priority=1) → testLogin()
3. @Test(priority=2) → testAddProductToCart()
4. @Test(priority=3) → testVerifyProductInCart()
5. @Test(priority=4) → testCheckoutFlow()
6. @AfterClass → Closes browser
